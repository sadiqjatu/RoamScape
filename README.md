# 🏝️ RoamScape

**RoamScape** is a full-stack P2P (peer-to-peer) rental marketplace where users can explore, list, and book accommodations like homes, rooms, and vacation rentals. Built with the **MERN stack (MongoDB, Express, React, Node.js)**, RoamScape provides a user-friendly platform for short-term stays.

---

## 🚀 Features

- 🔍 **Search Listings**: Browse stays by location, type, and price
- 🏠 **List a Property**: Users can create and manage rental listings
- 📅 **Booking System**: Request and confirm bookings
- 👤 **User Authentication**: Secure login/signup using passport (local strategy)
- 🖼️ **Image Upload**: Add photos for listings
- 📱 **Responsive UI**: Mobile and desktop-friendly interface

---

## 🛠️ Tech Stack

| Tech | Role |
|------|------|
| **MongoDB** | Database for storing users, listings, bookings |
| **Express.js** | Backend framework (API + routing) |
| **React.js** | Frontend SPA (Single Page Application) |
| **Node.js** | Backend runtime |
| **Mongoose** | ODM for MongoDB |
| **passport** | Authentication |
| **Cloudinary / Multer** | Image handling|
| **CSS** | Styling (based on your choice) |
